Virtual CPU (Architecture Project / Team) - Spring 2022

Worked in a team to develop a virtual CPU from boolean gates up.

  • Studied how high-level programs are converted to assembly and machine code and then executed on hardware.
  • Specified and parameterized functions, inputs and outputs for the units within the CPU for our team to accomplish a more seamless integration.
  • Worked individually as well as with team members to design and virtually build various CPU units.
  • Learned how CPU units were designed from boolean gates up but did use premade units (such as RAM) in our final project
